Project objective: Create a professional product management system to efficiently update product prices and group prices in a bulk operation.

How to use: 
1. Run the source code.
2. Choose an option from the menu that will appear in the console:
   - Update a product price.
   - Add a new product to the list.
   - Save changes.
   - Exit

What problem does it solve? 
The project addresses the issue of updating prices for groups of products that need to have the same value, without the need to modify each one individually, thus saving time and reducing the possibility of human error.
